ForRunning this project:
A makefile is given which will compile and create the object file for given java files.

command to run server:
Server Will run without any input because it has all the information.
server class: P2P SERVER

command to run client:
client with start with asking server address. For us we tried this on single machine only so localhost was given.This client will connect to server.
Now client can enter its hostname and upload port on which server is running.
after this client's upload server will start and client can use all commands like lookup add listall etc.

For getting RFC from other client client need to give "GET <RFC_NO.> <UPLOAD_PORT_NUMBER>".

and rfc will be downloaded to "s.txt" file. 
client class: P2P CLIENT
